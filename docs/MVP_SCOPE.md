# Realtime MVP Scope

## 1) 목표
- 개인 공부 타이머를 안정적으로 측정하고 복구한다.

## 2) 포함 기능
- 스플래시 -> 홈 -> 솔로 공부 화면 진입
- 카메라 기반 얼굴 상태 인식
- 공부 시간 누적/일시중지 로직
- 앱 재실행 시 세션 복구 (`SharedPreferences`)
- 기본 설정 화면(로그인, 타임랩스 토글 UI)

## 3) 제외 기능 (후순위)
- 그룹 스터디 실시간 동기화
- 초대 링크/채팅
- 광고/리워드
- 커스텀 아바타 이미지 에셋

## 4) 완료 기준
- `flutter analyze` 경고/에러 0
- `flutter test` 통과
- Android 디버그 실행 시 홈/솔로 화면 정상 동작
