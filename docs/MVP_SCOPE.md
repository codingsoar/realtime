# Realtime MVP Scope

## 1) 목표
- 개인 공부 시간을 안정적으로 측정하고 복구한다.

## 2) 포함 기능
- 스플래시 후 바로 공부 화면 진입
- 카메라 기반 얼굴 상태 인식
- 공부 시간 누적/일시중지 로직
- 세션 데이터 영속화(`SharedPreferences`)
- 기본 설정 화면(로그아웃, 타임랩스 관련 UI)

## 3) 제외 기능(후순위)
- 그룹 스터디 실시간 동기화
- 고급 랭킹/채팅
- 광고/리워드 및 커스텀 아바타/이미지 에셋

## 4) 완료 기준
- `flutter analyze` 경고/에러 0
- `flutter test` 통과
- Android 디버그 실행 시 초기 화면/공부 화면 정상 동작
